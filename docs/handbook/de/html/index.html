Zusammenfassung

BibleTime ist ein absolut freies Bibelarbeits-Programm. Seine Benutzeroberfläche ist mit dem Qt-Rahmenwerk erstellt, das es auf mehreren Betriebssystemen lauffähig macht, darunter Linux, Windows, FreeBSD und Mac OS X. Die Software nutzt die SWORD-Programmierbibliothek, um mit über 200 freien Bibeltexten, Kommentaren, Wörterbüchern und Büchern zu laufen, die von der Crosswire Bible Society zur Verfügung gestellt werden.
