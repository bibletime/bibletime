Tiivistelmä

BibleTime is a completely free Bible study program. The program's user interface is built with Qt framework, which makes it run on several operating systems including Linux, Windows, FreeBSD and Mac OS X. The software uses the SWORD programming library to work with over 200 free Bible texts, commentaries, dictionaries and books provided by the Crosswire Bible Society.
